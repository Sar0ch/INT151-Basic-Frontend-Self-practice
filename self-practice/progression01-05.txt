Week01 
-เรียนเรื่อง array เลยนำสิ่งที่ได้เรียนไปนั้น ให้ chatgpt ทำโจทย์ขึ้นมาให้ โดยที่ไล่ระดับ จาก ง่าย ไป ยาก 
-พยายามฝึกทำโดยไม่เปิดดู นอกจากจำไม่ได้จริงๆว่าต้องใช้ยังไง ถึงจะไปถามเฉพาะจุด
-เข้าใจการแทรก-ลบที่ array มากขึ้นโดยใช้ splice ว่าใช้งานอย่างไร 
-สามารถอธิบาย logic เรื่อง array ให้เพื่อนเข้าใจได้มากขึ้นหลังจากได้ลองฝึกทำด้วยตัวเอง

Week02
-เข้าใจความต่างระหว่าง value vs reference อย่างชัดเจน (numbers/strings เทียบค่าตรง ๆ, objects เทียบที่อยู่ในหน่วยความจำ) ->ทำให้เวลาเจอโจทย์เกี่ยวกับการเปรียบเทียบ object ไม่งง
-รู้จักการ ออกแบบโครงสร้าง object ด้วย class เพื่อให้โค้ด reusable 
-Destructuring ทำให้มอง object ซ้อน (nested object) ง่ายขึ้น เช่น student.address.city → ใช้ { address: { city } } ดึงได้เลย

Week03
-เข้าใจการใช้ arrow function มากขึ้น
-ได้ฝึกใช้กาเรียกใช้ function 

Week04
-เข้าใจการใช้ array method แต่ละอย่างว่าใช้ยังไงแต่ บางอย่างอาจจะยังไม่เข้าใจในตอนแรก แต่พอได้ฝึกทำโจทย์มากขึ้นก็ได้เข้าใจมากขึ้น
-คำความเข้าใจ method ต่างๆว่าใช้ในสถานการณ์ไหน การนำมาใช้ต่างๆ

Week05
- เข้าใจการใช้ ES module รู้จัก Tree shakeing มีบางส่วนที่ยังไม่เข้าใจบ้างในห้อง แต่กลับมาย้อนดูคลิปแล้วฝึกทำ self practice ก็ช่วยให้เข้าใจมากขึ้น
- ได้ลองใช้ method มากขึ้นจนเริ่มใช้คล่องขึ้น
